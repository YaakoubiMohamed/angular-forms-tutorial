Référence Rapide API

Classes

Classe Module Description
FormControl @angular/forms Suit la valeur et la validation d'un contrôle individuel
FormGroup @angular/forms Suit la valeur et la validité d'un groupe de contrôles
FormArray @angular/forms Suit la valeur et la validité d'un tableau de contrôles
FormBuilder @angular/forms Service pour créer des instances de contrôles
Validators @angular/forms Fournit des fonctions de validation intégrées

Directives

Directive Sélecteur Description
FormGroupDirective [formGroup] Lie un FormGroup à un élément DOM
FormControlName formControlName Synchronise un FormControl avec un input
NgModel [(ngModel)] Directive de liaison bidirectionnelle
NgForm form:not([ngNoForm]) Crée un FormGroup de niveau supérieur

Propriétés

Propriété Type Description
value any Valeur actuelle du contrôle
valid boolean Le contrôle passe toutes les validations
invalid boolean Le contrôle échoue à une validation
pristine boolean L'utilisateur n'a pas modifié la valeur
dirty boolean L'utilisateur a modifié la valeur
touched boolean Le contrôle a reçu puis perdu le focus
untouched boolean Le contrôle n'a jamais perdu le focus
pending boolean Validation asynchrone en cours
status FormControlStatus Statut de validation
errors ValidationErrors | null Erreurs de validation

Observables

Observable Émet Description
valueChanges T Émet lors du changement de valeur
statusChanges FormControlStatus Émet lors du changement de statut

Définitions de Types

// Statut du contrôle de formulaire
type FormControlStatus = 'VALID' | 'INVALID' | 'PENDING' | 'DISABLED';

// Erreurs de validation
type ValidationErrors = { [key: string]: any } | null;

// Signature de fonction de validation
type ValidatorFn = (control: AbstractControl) => ValidationErrors | null;

// Signature de fonction de validation asynchrone
type AsyncValidatorFn = (control: AbstractControl) => 
  Promise<ValidationErrors | null> | Observable<ValidationErrors | null>;

Imports de Modules

Configuration Reactive Forms

import { ReactiveFormsModule } from '@angular/forms';
import { 
  FormControl, 
  FormGroup, 
  FormArray, 
  FormBuilder,
  Validators 
} from '@angular/forms';

// Dans un NgModule
@NgModule({
  imports: [ReactiveFormsModule]
})

// Dans un Standalone Component
@Component({
  standalone: true,
  imports: [ReactiveFormsModule]
})

Configuration Template-Driven Forms

import { FormsModule } from '@angular/forms';

// Dans un NgModule
@NgModule({
  imports: [FormsModule]
})

// Dans un Standalone Component
@Component({
  standalone: true,
  imports: [FormsModule]
})

Champs dynamiques ✅ Excellent ⚠️ Limité
Tests unitaires ✅ Facile ⚠️ Nécessite le DOM
Validation croisée ✅ Native ⚠️ Complexe
Validation asynchrone ✅ Native ⚠️ Limitée
Performance (OnPush) ✅ Optimisée ⚠️ Standard
Courbe d'apprentissage Plus raide Plus facile
Code boilerplate Plus Moins
Typage fort ✅ Fort ⚠️ Faible
Réutilisabilité ✅ Haute ⚠️ Basse
